# Gold Shop Website

This is the frontend for the Gold Shop e-commerce website.

## Deployment Status
![GitHub Pages Deployment](https://github.com/nas23ise-dot/gold-shopy/workflows/Deploy%20to%20GitHub%20Pages/badge.svg)

## Technologies Used
- Next.js 15
- React 19
- Tailwind CSS v4
- TypeScript
- MongoDB (backend)

## Access
- Website: https://nas23ise-dot.github.io/gold-shopy/
- Backend API: https://gold-shopy.onrender.com/api

## Setup
1. Clone the repository
2. Navigate to `consumer-website` directory
3. Run `npm install`
4. Run `npm run dev` for development
5. Run `npm run build` for production build
